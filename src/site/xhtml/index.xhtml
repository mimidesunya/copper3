Copper PDF オープンソースプロジェクト

HTML-PDF変換ミドルウェアCopper PDF のオープンソース版です。

公開したソースの利用条件は、商用利用しやすいApache License, Version 2.0としています。

ソースの取得とビルド

ソースはSubversionで公開しています。 アクセス方法はこちらをご覧ください。

ソースのビルドにはApache Maven2が必要です。 例えばUbuntuでは次のコマンドでインストールしてください。 

sudo apt-get install maven2

ソースを取得した後、copperディレクトリ内で

mvn package

を実行することで全てのプロジェクトをビルドすることができます。 JARは各プロジェクトのtargetディレクトリに生成されます。 ライブラリが必要とする全てのJARを取得するには

mvn package dependency:copy-dependencies

を実行してください。全てのJARが各プロジェクトのtarget/dependencyに置かれた状態になります。

主なプロジェクト

サンプルコード等はリンク先をご覧ください。

Sakae
縦書き対応のPDF出力ライブラリです。 Graphics2Dへの出力をPDF化できること、高速で省メモリなのが特徴です。
Homare
ページ分割に適したHTML/CSSレンダリングエンジンです。
