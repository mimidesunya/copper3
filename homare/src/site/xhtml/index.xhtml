説明

Homareはページ分割に適したHTML/CSSレンダリングエンジンです。 CSS2.1サポートし、CSS3の一部を実装しています。

使用方法

ソースを取得し、 

mvn package dependency:copy-dependencies

でビルドした後、homare/targetおよびhomare/target/dependencyに生成されたjarをアプリケーションにリンクしてください。

ドキュメント

現在、ドキュメントは充実しているとは言えませんが、以下のサンプルプログラムと、 Copper PDFのドキュメント がいくぶんか役に立つと思います。
